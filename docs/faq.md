# Cylix Demo FAQ

This FAQ covers common questions from **prospective buyers and MSSPs** reviewing the Cylix demo materials.

---

## General

### What is Cylix?

Cylix is an **AI-driven cybersecurity and purple-team validation platform** that helps organizations continuously test defenses, prioritize threats, and communicate risk.

### Is this open source?

No. Cylix is a **commercial platform**. This repository only provides **public demo assets** for evaluation and marketing.

### Does this repo include production code?

No. There is **no production source code** in this repository. It contains:

- Screenshots generated from mock data
- High-level overview and FAQ
- Content intended for public demos and buyers

## MSSP & Partner Questions

### Is Cylix designed for MSSPs?

Yes. Cylix is built with **multi-tenant, repeatable workflows** in mind:

- Standardized **purple-team and validation playbooks**
- Consistent reporting for many clients
- AI assistance to keep analyst workloads manageable

Cylix is designed to work alongside existing security tools and telemetry sources. Integration specifics are covered in **private technical documentation** and discussed during evaluation.
